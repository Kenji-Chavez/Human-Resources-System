# Sistema de Recursos Humanos

Proyecto CRUD de gestión de empleados desarrollado con Spring Boot y React.

## Funcionalidades
- Listar empleados
- Agregar empleados
- Editar empleados
- Eliminar empleados

## Tecnologías
Backend:
- Spring Boot
- Java
- MySQL

Frontend:
- React
- JavaScript
- CSS

## Estructura del proyecto
/backend-spring → API REST  
/frontend-react → Interfaz de usuario  

## Cómo ejecutar el proyecto

### Backend
cd backend-spring  
mvn spring-boot:run

### Frontend
cd frontend-react  
npm install  
npm start
